HTML

HTML (HyperText Markup Language): It provides the basic structure of web pages, organizing content into headings, paragraphs, links, and other elements to define how information is displayed.

CSS

CSS (Cascading Style Sheets): This controls the presentation of web pages, including layout, colors, fonts, and responsiveness, ensuring that sites look attractive and adapt well across devices.

JavaScript

JavaScript: A programming language that brings websites to life by adding interactivity, animations, and dynamic content, allowing users to engage with elements like forms, buttons, and media.
